Instrucciones para el tablero de juego

Instrucciones

1. Tablero por donde avanzaran los fichas de los jugadores

2. Cuando el sol llegue al fondo, se pierde el juego

3. Se pueden ver las manos de los otros jugadores

4. De aqui se obtienen mas cartas cada turno

5. Aqui se selecciona la carta ha usar en tu turno

6. En tu turno si seleccionas antes de que se acabe el contador, obtendras una ventaja

7. Al llenarse el Sun-Counter podras disminuir el avance del sol
